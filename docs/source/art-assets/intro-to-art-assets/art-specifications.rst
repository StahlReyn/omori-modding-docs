Art Specifications
==================

Image Resolutions
-----------------

+--------------------+------------+-----------------------------------------+
| Type               | Resolution | Notes                                   |
+====================+============+=========================================+
| Game Screen        | 640x480    | Most pictures and cutscene frames are   |
|                    |            | in this resolution                      |
+--------------------+------------+-----------------------------------------+
| Battle Animation   | 192x192    |                                         |
| (per frame)        |            |                                         |
+--------------------+------------+-----------------------------------------+
| Portraits          | 106x106    |                                         |
+--------------------+------------+-----------------------------------------+
| Tiles              | 32x32      |                                         |
+--------------------+------------+-----------------------------------------+

General Tips
------------

- For drawn assets, usually it is a good idea to draw at larger
  resolution first, then resized down later.

  - Average Color Interpolation + Sharpen Filter
