Installing Mods
===============

Installing OneLoader
--------------------

In order to run and use Omori mods, a mod loader must first be installed
to the game. The most commonly used mod loader is
`OneLoader <https://github.com/rphsoftware/OneLoader/releases>`__, which
supports nearly every mod for the game. To install it, download the
latest release zip to the ``OMORI`` folder on Steam then extract the zip there.
When prompted to overwrite files, select "Yes to All".

.. hint::
    If you're having trouble finding ``OMORI`` folder, go to the Local Files tab of the Properties
    window and click Browse Local Files.

Installing Mods
---------------

To install an Omori mod, copy or download the zip file or folder of the
mod to the ``OMORI/www/mods`` folder. The above video also walks through
that process if you're confused.

OMORI installed from steam, the game folder are likely to be in
``SteamLibrary\steamapps\common\OMORI`` or similar, depending on
location of where you installed the game.
