Omori Text Color Convention
===========================

Omori tends to have a color for specific type of items, which are
labelled to give emphasis. This is done with the color code command of
/c[n] as mentioned in Text Codes section of this wiki. 

Color Source
------------

The color picked are from Windows.png file, which is found in
img/system/Windows.png

In the Windows.png file, there is group of color pallette on the bottom
right corner. The color picked are referenced from the color in the
square. Starting from first square being number 0.

{% hint style="info" %} Note that Omori changes the Windows.png color
palette from the usual standard default {% endhint %}
